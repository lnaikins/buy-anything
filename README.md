# Buy Anything

An e-commerce app UI.

## Prerequisite

To build this project you require:

- Android Studio Koala 2024.1.1 or higher
- Gradle 8.4.2+
- Kotlin 2.0.0+


# Technologies Used

- [Jetpack Compose](https://developer.android.com/develop/ui/compose): Android's modern UI Toolkit
